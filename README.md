Welcome to my **50 Projects in 50 Days** repository! 

After completing the 30-Day Vanilla JS Challenge, I am continuing my frontend development journey by building 50 unique, interactive mini-applications in 50 consecutive days. This repository tracks my daily progress and serves as a portfolio of modern UI/UX design, CSS animations, and DOM manipulation.

This challenge is based on the fantastic [50 Projects In 50 Days - HTML, CSS & JavaScript](https://www.udemy.com/course/50-projects-50-days/) course created by **Brad Traversy** and **Florin Pop** on Udemy.

---

## 🎯 Learning Objectives
- **Advanced CSS Architecture:** Mastering Flexbox, CSS Grid, custom properties (variables), and responsive design.
- **Micro-Interactions:** Building smooth CSS transitions, keyframe animations, and complex hover states.
- **Vanilla DOM Manipulation:** Selecting elements, updating classes, managing event listeners, and manipulating the DOM tree dynamically without frameworks.
- **Consistency & Discipline:** Writing clean, scalable, and professional code every single day.

## 💻 How to Run Locally

If you want to clone this repository and test these projects on your local machine:

1. Clone the repo to your local machine:
   `git clone https://github.com/ashishbtech/50-Projects-50-Days.git`
2. Navigate into any specific project directory.
3. Open `index.html` in your browser, or use an extension like **Live Server** in VS Code for hot-reloading.
